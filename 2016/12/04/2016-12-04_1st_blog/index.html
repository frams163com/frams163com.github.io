我的第一篇博客

早就想搭一个自己的博客，写点自己这些年折腾计算机的经历，今天算是完成愿望了。

其实实验室里头最想搭博客的不是我，是我们最小的师妹–如果从五师妹算起，她应该算七师妹。

实验室还有一个师弟，叫谭啸泽，我不知道泽的拼音是zhe还是ze，拼音输入法打到 tanxiaoz 我就卡住了，随便选了提示里的第一个，然后他就成了谭校长。

有一天谭校长告诉小师妹阿里卖域名可便宜了1块钱一个，她就买了，结果发现光有域名还搭不了自己的博客网站，于是她又在谭校长的推荐下买了万网虚拟主机，结果发现还是不行，因为万网虚拟主机不能绑定到她的.xyz域名。不过还好号称全球最大的程序员交友网站 github 提供了静态页面服务，搭个博客还是够的。如果你也想搭博客，欢迎访问github首页 ┋◆冃.狌.交.伖，释.鲂.压.劦、棑.解.漃.瘼◆ 真 人】视||频. █网.址：github.com/◆┋[1]，网址绝对真实。

谭校长的坑就这么挖了，挖了也就挖了，偏偏有人往里跳，跳了也就跳了，偏偏是我来填这个坑。我为啥往坑里跳，我想一定是我心肠软，不是我傻，一定不是我傻。

可是搭博客我也不会呀，git 有三四十个命令，我只会 init、add、commit、push，滚回上个版本还得查谷歌；node.js、npm、Hexo、Jekyll 更是听都没听过，更别提各种好看的博客主题了。

作为技术宅，不会我也敢上。npm 不就 package 管理工具嘛，当 apt 用就行; hexo g 不就是把 markdown 文件变成 html 文件嘛，当 gcc 用好了; git 有三四十个命令一个也没用上，hexo d一键部署，自动完成 git add、commit、push; … 当然建博客主要是师妹搞定的，我只负责一点点的技术支持。

经过了三天的努力，师妹的博客总于发布了，这时候我不要脸地要了一个拷贝，用

替换了里面所有的 likelly, 然后就成了我的博客，替换可能不完全，如果你在我的博客中看到 likelly，请自行脑补成 frams，如果你有强迫症，不改不行，请告诉我。

虽然不是很光彩，但我的博客就是怎么来的。
